Website Planning Document

Site Name

Name: Coastal Crafters

Description: This name represents a small business focused on handmade coastal-themed crafts, such as seashell decor and driftwood art, targeting coastal lifestyle enthusiasts. The name is simple, memorable, and reflects the business's focus on creativity and coastal aesthetics.

Optional Domain Availability: coastal-crafters.com

Site Purpose

The Coastal Crafters website serves as an online storefront to showcase and sell handmade coastal-themed crafts. It provides product listings, a shopping cart, and a contact form for custom orders, along with a blog featuring crafting tips and coastal lifestyle inspiration.

Scenarios

  • What types of coastal-themed crafts are available for purchase?
  • How can I contact the business to request a custom seashell decor piece?

Color Schema

The color schema is inspired by coastal aesthetics:

  • Sandy Beige (#F5F5DC): Used for the background to evoke a beach-like feel.
  • Sea Blue (#4682B4): Used for headings and accents to reflect the ocean.

Typography

The typography is clean and modern, ensuring readability and aesthetic appeal:

  • Lora (Serif): Used for headings to convey elegance and craftsmanship.
  • Open Sans (Sans-Serif): Used for body text for clarity and readability.

Wireframe

Mobile View

The mobile homepage layout is a single-column design:

  • Header: Logo at the top, followed by a collapsible navigation menu (Home, Shop, Blog, Contact).
  • Main Content: A hero image of coastal crafts, a brief welcome message, and a featured products section (3 items in a vertical list).
  • Footer: Social media links and contact info.

Desktop View

The desktop homepage layout uses a wider, multi-column design:

  • Header: Logo on the left, horizontal navigation bar on the right.
  • Main Content: A large hero image spanning the width, a welcome message in the center, and a featured products section (3 items in a horizontal grid).
  • Footer: Three columns for About, Contact, and Social Media links.

Note: Wireframe sketches should be created separately using a tool like Figma or hand-drawn on paper, showing the layout described above.

Testing

Self-check the site plan document using the following tools:

  • W3C Validator: Ensure HTML and CSS are valid.
  • WAVE: Check for accessibility issues (e.g., alt text, contrast).
  • Lighthouse: Evaluate performance, SEO, and best practices.
  • Color Contrast Checker: Verify that Sandy Beige (#F5F5DC) and Sea Blue (#4682B4) meet WCAG contrast ratios.
